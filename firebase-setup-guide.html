‚è≥ Next: Firebase Project Setup

Follow these steps to create your free Firebase project:

  1. Go to Firebase Console: console.firebase.google.com
  2. Create New Project: Click "Create a project"
  3. Project Name: "phenohunter-alpha"
  4. Google Analytics: Optional (can skip for alpha)

üìù Firebase Configuration Steps

1. Enable Authentication:

  • Go to Authentication ‚Üí Sign-in method
  • Enable "Email/Password"
  • Disable "Email link (passwordless sign-in)"

2. Create Firestore Database:

  • Go to Firestore Database ‚Üí Create database
  • Start in "Test mode" (we'll secure it later)
  • Choose location closest to your users

3. Enable Storage:

  • Go to Storage ‚Üí Get started
  • Start in "Test mode"
  • Use same location as Firestore

4. Get Configuration:

  • Go to Project Settings (gear icon)
  • Scroll to "Your apps" section
  • Click "Web app" icon (</>)
  • App nickname: "phenohunter-web"
  • Copy the configuration object

üîß Update Local Configuration

Update your .env.local file with the Firebase config values:

VITE_FIREBASE_API_KEY=your-actual-api-key VITE_FIREBASE_AUTH_DOMAIN=phenohunter-alpha.firebaseapp.com VITE_FIREBASE_PROJECT_ID=phenohunter-alpha VITE_FIREBASE_STORAGE_BUCKET=phenohunter-alpha.appspot.com VITE_FIREBASE_MESSAGING_SENDER_ID=your-sender-id VITE_FIREBASE_APP_ID=your-app-id

üöÄ Vercel Deployment Steps

  1. Go to Vercel: vercel.com
  2. Sign up: Use GitHub account (free)
  3. Import Git Repository: Connect your GitHub repo
  4. Deploy: Click "Deploy" (uses automatic settings)
  5. Environment Variables: Add your Firebase config to Vercel dashboard
